# Getting Started

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.

## Key Features

### 🎨 Dual Modes
- **Elegante**: Beige, Crema, Verde Salvia, Terracota Suave
- **Salvaje**: Naranjas Vibrantes, Rojos, Cobres

### ✈️ Journey Narrative
- 16 ciudades del mundo
- Insights y lecciones de diseño
- Interactividad en hover

### 💰 Dynamic Services Calculator
- Selección individual de servicios
- Precio en tiempo real
- Post y Reel selectors

### ⏱️ Express 24hrs Counter
- Contador en tiempo real
- Widget flotante

### 🎵 Wild Mode
- Spotify playlist integrada
- Modo "party" explosivo
- Partículas animadas

### 📱 Contact to WhatsApp
- Formulario minimalista
- Envío directo a WhatsApp
- +56939548475
