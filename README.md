# Portfolio Alexis Negro Calderón

## Brand Powering Designer - Immersive Web Experience

Landing page épica y minimalista que cuenta la historia de un emprendedor que transformó su experiencia como flight attendant en creatividad para diseñar experiencias web memorables.

### Features

✈️ **Narrativa de Viaje** - 16 ciudades del mundo con historias y insights de diseño
🎨 **Modo Dual** - Elegancia minimalista + Modo Salvaje (Party Vibes)
🎵 **Spotify Integrado** - Playlist en modo salvaje
💰 **Configurador Dinámico** - Selector de servicios con calculadora en tiempo real
⏱️ **Contador 24hrs** - Express Site Experience
📱 **Formulario → WhatsApp** - Contacto directo
✨ **Animaciones Smooth** - Framer Motion para experiencia VR

### Stack

- **Next.js 14** - Framework de React
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animaciones
- **Zustand** - State management

### Desarrollo

```bash
npm install
npm run dev
```

Visita `http://localhost:3000`

### Build

```bash
npm run build
npm start
```
